# 🎨 АРХИТЕКТУРА RadarTcell (КРАТКО)

## ✅ СЛОИ

```
HTTP → Handler → Service → Repository → PostgreSQL
```

- **Handler**: читает HTTP, формирует ответ.
- **Service**: бизнес‑логика (координаты, фильтры, сортировки).
- **Repository**: SQL и работа с БД.

---

## ✅ ОСНОВНЫЕ СУЩНОСТИ
