# 🗺️ ИНТЕЛЛЕКТ‑КАРТА RadarTcell (КРАТКО)

## ✅ Что это

REST API для каталога технологий. Сервер считает координаты, фильтрует, сортирует и отдает готовые данные фронту.

---

## ✅ Слои

- HTTP: `router.go`, handlers.
- Service: бизнес‑логика.
- Repository: SQL и БД.

---

## ✅ Ключевые сущности

Technology, Trend, SDG, Tag, Organization, MetricDefinition, Preferences.

---

## ✅ Публичные API

- `/api/technologies`, `/api/technologies/{slug}`
- `/api/trends`, `/api/sdgs`, `/api/tags`, `/api/organizations`, `/api/metrics`
- `/api/*/{slug|code}/technologies`
- `?locale=ru`

---

## ✅ Админ API

- `/api/admin/*` (CRUD)
- `/api/admin/i18n/*` (переводы)

---

## ✅ Системные

- `/healthz`, `/readyz`, `/metrics`, `/openapi.yaml`, `/swagger/`

---

## ✅ Мониторинг и логи

- Prometheus `/metrics`
- Grafana `http://localhost:3000`
- Zap JSON‑логи
