# 📖 ДОКУМЕНТАЦИЯ ПРОЕКТА RadarTcell

## 🎯 Добро пожаловать!

Это полная документация проекта **RadarTcell** — REST API для управления каталогом технологий на Go + PostgreSQL.

**Проект использует:**
- Go 1.24
- PostgreSQL 15
- Clean Architecture
- REST API
- JWT авторизация
- Swagger документация
