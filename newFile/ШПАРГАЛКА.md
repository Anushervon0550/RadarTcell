# 📋 ШПАРГАЛКА: RadarTcell

## ✅ БЫСТРЫЙ СТАРТ

```powershell
cd C:\PROJECT\RadarTcell\deploy
docker compose up -d

cd C:\PROJECT\RadarTcell
migrate -path "./migrations" -database $env:DATABASE_URL up

go run .\cmd\api\main.go
```

PUT    /api/admin/technologies/{slug}
