# 📋 ШПАРГАЛКА: RadarTcell

## 🎯 ЧТО ЭТО?

**RadarTcell** — REST API для каталога технологий на Go + PostgreSQL.

---

## 📁 ГЛАВНЫЕ ПАПКИ

| Папка | Что там |
|-------|---------|
| `cmd/api/` | **Старт** приложения (`main.go`) |
| `internal/domain/` | **Модели** данных (структуры) |
| `internal/ports/` | **Интерфейсы** (контракты) |
| `internal/service/` | **Бизнес-логика** |
| `internal/repository/` | **SQL** запросы |
| `internal/httpapi/` | **HTTP** обработчики |

// internal/service/technology_service.go
func (s *TechnologyService) List(ctx context.Context, params domain.TechnologyListParams) ([]domain.TechnologyListItem, int, error) {
    // Валидация
    if params.Page < 1 { params.Page = 1 }
    if params.Limit < 1 || params.Limit > 100 { params.Limit = 20 }
    
    // Вызов репозитория
    return s.repo.FindAll(ctx, params)

### Handler (HTTP обработчик)

```go
// internal/httpapi/technology_handler.go
func (h *TechnologyHandler) List(w http.ResponseWriter, r *http.Request) {
    // Парсим параметры
    params := domain.TechnologyListParams{
        Search: r.URL.Query().Get("search"),
        Page:   parseIntQuery(r, "page", 1),
        Limit:  parseIntQuery(r, "limit", 20),
    }
    
    // Вызываем сервис
    items, total, err := h.service.List(r.Context(), params)
    if err != nil {
        writeError(w, http.StatusInternalServerError, "failed to load technologies")
        return
    }
    
    // Возвращаем JSON
    writeJSON(w, http.StatusOK, map[string]any{
        "data":  items,
        "total": total,
    })
}
```

## 🔧 ПОЛЕЗНЫЕ ФУНКЦИИ

### Парсинг query параметров

```go
// internal/httpapi/request_helpers.go
func parseIntQuery(r *http.Request, key string, def int) int {
    val := r.URL.Query().Get(key)
    if val == "" {
        return def
    }
    n, err := strconv.Atoi(val)
    if err != nil {
        return def
    }
    return n
}
```

### Отправка JSON

```go
// internal/httpapi/json.go
func writeJSON(w http.ResponseWriter, status int, data any) {
    w.Header().Set("Content-Type", "application/json")
    w.WriteHeader(status)
    json.NewEncoder(w).Encode(data)
}

func writeError(w http.ResponseWriter, status int, message string) {
    writeJSON(w, status, map[string]string{"error": message})
}
```

## 🎓 АРХИТЕКТУРНЫЕ ПАТТЕРНЫ

### 1. Clean Architecture

```
HTTP API (внешний слой)
    ↓
Service (бизнес-логика)
    ↓
Repository (работа с БД)
    ↓
PostgreSQL (внешний слой)
```

### 2. Dependency Injection

```go
// Все зависимости передаются через конструкторы
service := NewTechnologyService(repo)
handler := NewTechnologyHandler(service)
```

### 3. Repository Pattern

```go
// Интерфейс скрывает детали БД
type TechnologyRepository interface {
    FindAll(ctx, params) ([]Technology, error)
}

// Можем подменить реализацию (для тестов, другая БД)
```

### 4. Middleware

```go
// Цепочка обработчиков
r.Use(middleware.Logger)
r.Use(middleware.Recoverer)
r.Use(AuthMiddleware)
```
